# Smart Customer Churn Prediction System

## Project Overview
This project predicts whether a telecom customer is likely to churn (leave the service) using Machine Learning.  
The system analyzes customer-related features such as tenure, monthly charges, and support calls to predict churn.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning (Random Forest Classifier)
- VS Code

## Features
- Customer churn prediction
- Data preprocessing
- Model training and testing
- Accuracy evaluation
- User input-based prediction

## Project Workflow
1. Data Collection  
2. Data Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Model Training  
6. Prediction Generation  

## How to Run
1. Install required libraries:
```bash
pip install pandas numpy scikit-learn
```

2. Run the Python file:
```bash
python SmartCustomerChurnPrediction.py
```

## Sample Input
- Customer Tenure: 12  
- Monthly Charges: 90  
- Support Calls: 4  

## Sample Output
Prediction: Customer will churn
